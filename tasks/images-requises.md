## CONSEILS POUR LES PHOTOS

1. **Meme angle avant/apres** : C'est LE point le plus important. Prendre la photo "avant" depuis un endroit precis, noter l'emplacement, et reprendre l'exacte meme photo apres intervention.

2. **Lumiere** : Eviter les contre-jours. Le soleil dans le dos du photographe est ideal. Les photos "avant" par temps gris et "apres" par beau temps renforcent le contraste (mais pas obligatoire).

3. **Resolution** : Minimum 1200px de large. Les photos seront affichees en 4:3 donc privilegier le format paysage.

4. **Authenticite** : Pas de photos de stock. Les visiteurs Meta Ads detectent les photos generiques. Des vraies photos de vrais chantiers dans le Loiret sont beaucoup plus convaincantes.

5. **Nommage** : Utiliser les noms de fichiers suggeres ci-dessus pour faciliter l'integration.

6. **Variete** : Varier les types de toiture (tuiles terre cuite, tuiles beton, ardoises) et les types de maison (pavillon, maison de ville, villa) pour que chaque visiteur puisse s'identifier.

7. **Meteo et saison** : Photos de printemps/ete ideales (vegetations, ciel bleu). Eviter les photos avec arbres nus ou ciel gris uniforme pour les "apres".
