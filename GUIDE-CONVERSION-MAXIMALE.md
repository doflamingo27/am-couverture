# PARTIE 1 : LES 10 RÈGLES D'OR POUR DES LEADS DE QUALITÉ

### Règle 1 — Le hook décide de tout
Les 125 premiers caractères sont les seuls que 80% des gens liront. Le hook doit faire 3 choses en 1 seconde : identifier la cible, nommer le problème, promettre une solution. Si le hook est faible, le reste du texte ne sera jamais lu.

### Règle 2 — Primary text court (< 125 caractères visibles)
Meta n'affiche que les ~125 premiers caractères avant le "voir plus". Tout ce qui compte doit être AU-DESSUS de la coupure. Le reste du texte sert à convaincre ceux qui cliquent "voir plus" — c'est du bonus, pas l'essentiel.

### Règle 3 — Qualifier dans le texte pour filtrer les curieux
Inclure des éléments qui éliminent les mauvais leads : "Propriétaires dans le Loiret", "Votre toiture a plus de 10 ans ?". Les gens qui ne se reconnaissent pas ne cliquent pas = moins de clics inutiles = CPL qualifié plus bas.

### Règle 4 — Preuve sociale immédiate
Intégrer des signaux de confiance directement dans le texte : "4.9/5 sur Google", "Plus de 200 toitures traitées", "Garantie décennale". Les propriétaires dans le BTP ont peur des arnaques — la confiance doit être dans chaque phrase.

### Règle 5 — Un seul CTA clair
Ne pas diluer l'attention avec plusieurs actions possibles. Chaque pub = UN objectif = UN CTA. "Demandez votre devis gratuit" ou "Réservez votre diagnostic". Jamais les deux en même temps.

### Règle 6 — Headline = bénéfice concret, pas slogan
Le headline (titre sous l'image) doit résumer le bénéfice concret en une phrase. Pas de "AM Couverture, expert toiture". Oui à "Démoussage + Hydrofuge — Résultat visible immédiatement".

### Règle 7 — Description = urgence + local
La description (sous le headline) est le dernier texte lu avant le clic. C'est l'endroit pour l'urgence et la localisation : "Places limitées ce mois-ci · Artisan local Loiret (45)".

### Règle 8 — Pas de questions dans le hook (pour le local service)
Les questions ("Votre toit est-il propre ?") performent moins bien que les affirmations directes ("Votre toit se dégrade en silence") pour les services locaux. L'affirmation crée un sentiment d'urgence, la question laisse la place au doute.

### Règle 9 — Format PAS (Problem-Agitate-Solve)
La structure la plus efficace pour les services locaux :
- P = Problème (nommer la douleur)
- A = Agiter (amplifier les conséquences)
- S = Solution (ta prestation = le soulagement)

### Règle 10 — Landing page = prolongement exact de la pub
Si la pub parle de diagnostic gratuit, la LP doit avoir "Diagnostic gratuit" comme titre H1 visible en premier. Toute dissonance entre la pub et la LP tue la conversion. La promesse doit être tenue immédiatement.

# PARTIE 3 : OPTIMISATION DES LANDING PAGES

## A. Structure de la page — Ordre des sections par impact

L'ordre optimal testé pour les services BTP local :

```
1. HERO + FORMULAIRE (above the fold)
   → Titre H1 = promesse exacte de la pub
   → Formulaire visible sans scroller
   → Bouton CTA orange/chartreuse

2. BARRE DE CONFIANCE
   → 4 badges : Devis gratuit en 24h | Décennale | 4.9/5 Google | Artisan local

3. AVANT / APRÈS
   → 2-3 photos comparatives
   → Le visuel le plus convaincant de toute la page

4. COMMENT ÇA MARCHE (3-4 étapes)
   → Processus simple et rassurant
   → "1. Vous demandez → 2. On inspecte → 3. On traite → 4. Résultat garanti"

5. SERVICES INCLUS
   → Liste de tout ce qui est compris
   → Nettoyage, traitement, hydrofuge, gouttières, rapport photos

6. AVIS CLIENTS
   → 3-4 vrais témoignages avec prénom + ville
   → Étoiles jaunes visibles

7. FAQ
   → 5 questions qui traitent les objections
   → "C'est cher ?" / "Arnaque ?" / "Kärcher ?" / "Durée ?" / "Bon moment ?"

8. CTA FINAL
   → Répétition du formulaire
   → Urgence : "Places limitées ce mois-ci"
```

## B. Le formulaire — Règles de conversion maximale

### Ce qu'il faut :
- **Max 4-5 champs** : Prénom, Téléphone, Code postal, Type de projet (dropdown), Email (optionnel)
- **Multi-étapes** (déjà en place) : La première étape ne demande que le type de bien et le type de besoin. Le téléphone arrive à l'étape 2 — l'utilisateur est déjà engagé psychologiquement
- **Bouton CTA** : "Recevoir mon devis gratuit" (pas "Envoyer" qui est froid)
- **Texte sous le bouton** : "Rappel sous 2h · Sans engagement · 100% gratuit"
- **Badge de confiance** à côté du formulaire : cadenas + "Vos données ne sont jamais partagées"

### Ce qu'il faut éviter :
- Ne PAS demander l'adresse complète (friction trop élevée)
- Ne PAS demander le nom de famille (inutile pour un premier contact)
- Ne PAS mettre de captcha visible (tue la conversion)
- Ne PAS demander un message/commentaire libre (les gens ne savent pas quoi écrire)

### Astuce conversion +30% : le formulaire multi-étapes
Tu l'as déjà implémenté — c'est le meilleur pattern. L'utilisateur commence par une question facile ("Quel type de bien ?"), puis avance naturellement. Le taux d'abandon entre l'étape 1 et la dernière est beaucoup plus bas qu'un formulaire classique avec tous les champs visibles d'un coup.

## C. Optimisations techniques de la LP

### Vitesse (critique)
- Objectif : PageSpeed > 90 sur mobile
- Chaque seconde de chargement supplémentaire = -7% de conversion
- Images WebP/AVIF via next/image (déjà en place)
- Pas de bibliothèques JS lourdes

### Mobile-first (80%+ du trafic)
- Formulaire visible sans scroller sur mobile
- Bouton CTA assez large pour un pouce (min 48px de hauteur)
- Numéro de téléphone cliquable (tel:) dans la barre sticky
- Pas de pop-ups qui bloquent le contenu

### Barre sticky téléphone (déjà en place)
- Garder la barre sticky en bas de l'écran sur mobile
- Numéro cliquable = conversion directe sans formulaire
- Certains leads préfèrent appeler plutôt que remplir un formulaire

## E. Page /merci — Maximiser la valeur post-conversion

La page de remerciement est un moment d'attention maximale. Le lead vient de convertir, il est engagé. Profiter de cette page pour :

1. **Confirmer** : "Votre demande a bien été reçue. Un conseiller vous rappelle sous 2h."
2. **Rassurer** : Rappeler les badges de confiance (décennale, 4.9/5, artisan local)
3. **Préparer** : "Voici ce qui va se passer : (1) Rappel sous 2h (2) Rendez-vous d'inspection (3) Devis détaillé"
4. **Bonus** : "En attendant, voici 3 signes que votre toiture a besoin d'entretien" (contenu éducatif)
5. **Tracking** : C'est sur cette page que l'event Meta Pixel "Lead" se déclenche (déjà en place)

## F. Vitesse de rappel = facteur n°1 de conversion lead → client

Les données montrent que rappeler un lead dans les 5 premières minutes multiplie par 9 les chances de conversion par rapport à un rappel en 30 minutes. Après 24h, le lead est pratiquement mort.

**Action pour ton client** :
- Notification email immédiate via Resend (déjà en place)
- Il DOIT rappeler dans les 2h maximum — c'est promis dans les pubs
- Idéal : rappel dans les 15 minutes
- Script de rappel : "Bonjour [prénom], c'est [nom] d'AM Couverture. Vous avez demandé un devis pour [type de projet]. Quand est-ce que je peux passer voir votre toiture ?"

# PARTIE 4 : AMÉLIORATIONS TECHNIQUES À IMPLÉMENTER

### 1. UTM tracking sur chaque pub
Chaque URL de destination doit avoir des UTM :
```
https://www.am-couverture.net/demoussage?utm_source=facebook&utm_medium=cpc&utm_campaign=vague1&utm_content=p1-transformation
```
Ça permet de voir dans Google Analytics QUELLE pub génère quel trafic.

### 2. H1 dynamique selon utm_content
```javascript
// Dans le config.ts de chaque LP
const searchParams = useSearchParams();
const utmContent = searchParams.get('utm_content');

const headlines = {
  'p1-transformation': 'Votre toiture retrouve son éclat — Devis gratuit',
  'p2-diagnostic': 'Diagnostic toiture GRATUIT — On vous dit tout',
  'p3-comparatif': 'Protégez votre toit pour 10-20x moins cher',
  // etc.
};
```

### 3. Micro-interactions formulaire
- Animation subtile quand l'utilisateur passe à l'étape suivante
- Barre de progression visible ("Étape 2/3")
- Validation en temps réel (format téléphone)
- Le bouton change de texte : "Dernière étape →" puis "Recevoir mon devis gratuit"

### 4. Exit intent popup (desktop uniquement)
Quand le curseur quitte la page → afficher un popup : "Attendez ! Votre diagnostic toiture est GRATUIT et sans engagement. Laissez-nous juste votre numéro, on vous rappelle sous 2h."

### 5. Social proof en temps réel
Ajouter un bandeau discret : "Jean-Pierre de Fleury-les-Aubrais a demandé un devis il y a 2h" (peut être automatisé depuis le Google Sheet)

### 6. Conversion API (CAPI) feedback loop
Renvoyer à Meta les statuts "Devis signé" depuis le Google Sheet pour que l'algorithme Andromeda apprenne quels profils CONVERTISSENT VRAIMENT (pas juste qui remplit le formulaire). C'est la technique la plus puissante pour améliorer la qualité des leads à long terme.
